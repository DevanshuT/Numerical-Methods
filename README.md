# Numerical-Methods
This repositary contains the MATLAB implementation of main numerical methods like Newton-Raphson, Simpson's integration and numerical ODE solver.
1. Root Finder
	1. Bisection
	2. Euler
	3. Newton-Raphson
2. Linear Equations
	1. Gauss Elimination
	2. Gauss Seidel
	3. Thomas Algorithm
3. Numerical Integration
	1. Trapezoidal
	2. Simpson
4. ODEs and PDEs
	1. Finite Difference Method
	2. Explicit Euler
	3. Heuns-RungeKutta-Midpoint
	3. Crank Nicholson
	4. Implicit Euler
	5. Alternating Direction Implicit (ADI)
